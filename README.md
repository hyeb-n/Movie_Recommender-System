# 영화 추천 시스템 (Streamlit 앱)

사용자–영화 평점 데이터를 바탕으로 **협업 필터링·행렬 분해(SVD)** 로 평점을 예측하고, **사용자 군집(클러스터)** 별 추천 결과를 정리한 뒤, 그 결과를 **Streamlit 웹 앱**에서 탐색할 수 있게 한 프로젝트입니다.

## 프로젝트 구성

| 경로 | 설명 |
|------|------|
| `app.py` | Streamlit 기반 영화 추천 UI |
| `dm_model_2.ipynb` | 데이터 전처리, EDA, CF/SVD 학습, 클러스터링, `result_df`·`predicted_matrix` 생성 파이프라인 |
| `movie_genre_df.csv` | 영화 메타데이터: 제목, `movie_id`, ML-25k 스타일 장르 원-핫 컬럼 |
| `result_df.csv` | 클러스터별 **특화 추천 / 추천 / 비추천** 목록 (예상 평점, 특이성 점수 포함) |
| `predicted_matrix.csv` | 학습된 SVD로 채운 **사용자×영화 예측 평점 행렬** (용량이 큼) |
| `requirements.txt` | 앱 실행에 필요한 Python 패키지 |

## 모델링 (`dm_model_2.ipynb`)

노트북은 Google Colab에서 구동하도록 작성되어 있으며, 로컬에서는 데이터 경로만 맞추면 동일한 흐름으로 재현할 수 있습니다.

### 데이터

- 사용자×영화 **희소 평점 행렬**을 불러오고, 평가 수가 적은 사용자를 제거(분위수 기준 필터링)하여 분석·학습에 쓰는 부분 집합을 만듭니다.
- `movie_genre_df`와 결합하여 장르 분포 등 EDA를 수행합니다.

### 협업 필터링 & 행렬 분해

- **scikit-surprise**로 **user-based KNN**, **item-based KNN** 등 협업 필터링 실험을 합니다.
- 최종적으로 **SVD**(`n_factors=50`, 바이어스 항 사용)를 학습·검증(RMSE, MAE 등)하고, 관측되지 않은 셀까지 포함한 **완전한 예측 행렬**을 만듭니다. 이 결과가 `predicted_matrix.csv`의 원천 데이터입니다.

### 사용자 클러스터링 & `result_df`

- 예측 행렬을 **표준화**한 뒤 **PCA**(2차원)로 압축하고, **K-Means**(예: 클러스터 수 10)로 사용자를 군집화합니다.
- 군집별로 영화에 대한 **평균 예측 평점**을 구하고, 전체 평균·분산과 비교하여 **예상 평점**과 **특이성 점수**를 해석합니다.
- 이를 바탕으로 각 클러스터마다 **특화 추천**(군집에서 두드러지는 작품), **추천**(무난히 높은 예측), **비추천**(기대가 낮은 편) 등으로 정리한 표가 `result_df.csv`입니다.

> 노트북에서 생성한 CSV를 이 저장소 루트의 동일 파일명으로 두면, 앱이 그대로 읽어 사용합니다.

## 앱 (`app.py`)

**Streamlit** 단일 앱으로, 별도 회원가입 없이 CSV 기반 추천 결과를 둘러볼 수 있습니다.

### 기능 요약

1. **내 취향 추천**  
   좋아하는 영화를 고르고(제목 검색), 원하면 장르를 더 반영하거나 피하고 싶은 장르를 지정합니다. 선택한 영화들의 **장르 원-핫 벡터**를 합쳐 사용자 취향 벡터를 만들고, 카탈로그 영화와의 **유사도(내적)** 기준으로 추천합니다.

2. **클러스터 추천**  
   `result_df.csv`에 저장된 **군집 ID**를 고르고, **특화 추천 / 추천 / 비추천** 블록을 표로 확인합니다.

3. **장르로 찾기**  
   여러 장르를 동시에 만족하는 영화만 필터링하고, 제목 부분 검색으로 좁힐 수 있습니다.

4. **사용자 맞춤 (예측 행렬)**  
   `predicted_matrix.csv`에서 특정 `userId` 행을 읽어, **예측 평점 상위** 영화를 보여 줍니다. 파일이 크므로 앱에서 **필요할 때만** 로드하도록 되어 있고, 로드 후에는 세션 동안 캐시됩니다.

### 사이드바

**보고 싶은 목록(워치리스트)** 에 영화를 담아 두었다가, 각 탭에서 추천·검색 결과를 목록에 추가할 수 있습니다.

## 실행 방법

Python 3.10+ 권장.

```bash
cd /path/to/this/repo
python -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -r requirements.txt
streamlit run app.py
```

브라우저에서 표시되는 로컬 URL(기본 `http://localhost:8501`)로 접속합니다.

### 데이터 파일

- `predicted_matrix.csv`는 **수백 MB** 수준이 될 수 있어, 처음 로드 시 시간이 걸릴 수 있습니다.
- GitHub에는 용량 제한이 있으므로, 대용량 CSV는 [Git LFS](https://git-lfs.github.com/)나 별도 저장소·클라우드 저장 링크로 배포하는 방식을 검토하세요.

## 노트북 재현 시 참고

- Colab 원본은 `google.colab` 드라이브 마운트와 고정 경로를 사용합니다. 로컬에서는 해당 경로를 프로젝트의 `data/` 등으로 바꿉니다.
- Surprise 설치 환경에 따라 **NumPy 버전**을 맞출 필요가 있을 수 있습니다(노트북 상단의 pip 셀 참고). 앱 실행용 `requirements.txt`는 Streamlit·pandas·numpy만 포함합니다.
